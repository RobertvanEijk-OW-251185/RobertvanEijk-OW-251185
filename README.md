## About

I'm a frontend developer with a strong lean toward design — I care about the gap between "it works" and "it feels right." I spend most of my time in React and Figma, building interfaces that are both functional and considered.

Currently studying and shipping academic projects that span full-stack development, peer-to-peer platforms, and brand-driven web experiences.

## Current Projects

### 🛒 [Retro Rat](https://github.com/YOUR_USERNAME/retro-rat)
A peer-to-peer marketplace for vintage electronics built on the MERN stack. Features include user authentication, an admin console, a shopping cart, and role-based access control. Focused on security and a clean buyer/seller UX.

`MongoDB` `Express` `React` `Node.js`

`Figma` `Design Systems` `Scrollytelling`
