## About

I'm Robert, a Full Stack Development and UI/UX Design student at The Open Window.

My work sits at the intersection of development and design. I enjoy building products that are not only functional but intuitive, visually refined, and enjoyable to use. Whether I'm developing a MERN-stack application or designing a user journey in Figma, I'm interested in how thoughtful design decisions improve the overall user experience.

Most of my current work focuses on React, JavaScript, UI systems, and full-stack web development. Through both academic and personal projects, I've explored eCommerce platforms, peer-to-peer marketplaces, brand-driven web experiences, and responsive design systems.

Outside of development, I'm a competitive fencer at TuksFencing, where the discipline, strategy, and continuous improvement required by the sport strongly influence how I approach problem solving and design.

## Current Projects

### 🛒 [Retro Rat](https://github.com/RobertvanEijk-OW-251185/Retro-Rat)

A MERN-stack marketplace connecting buyers and sellers of vintage electronics.

Built with a strong focus on authentication, role-based permissions, marketplace workflows, and user experience. The platform includes product management, shopping cart functionality, admin moderation tools, and secure user accounts.

`MongoDB` `Express` `React` `Node.js`

### 🚌 WANATU

A brand-focused digital experience exploring how storytelling, motion, and interaction can communicate a transport company's identity online.

The project investigates scrollytelling, responsive design systems, and user-centred interface design through a high-fidelity Figma prototype.

`Figma` `UI/UX Design` `Design Systems` `Scrollytelling`

I believe good digital products are built where design and development meet.

A successful interface should not only function correctly—it should communicate clearly, guide users naturally, and create confidence through consistency. My goal is to build experiences that are technically sound, visually considered, and genuinely useful.
